Basic DOM Example

The list below has been created by JavaScript via the DOM.


const shopping = [
    'Apple',
    'Orange',
    'Strawberry',
    'Bananas',
    'Sausages'
];

const list = document.createElement('ul')

let innerHTML = ""
for (const item of shopping) {
   innerHTML += `<li>${item}</li>`
}

list.innerHTML = innerHTML
